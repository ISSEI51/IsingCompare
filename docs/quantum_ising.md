# 量子イジング模型と量子アニーリング

## 1. 問題ハミルトニアン H_p（古典イジング）

古典イジングモデルのエネルギー関数：

\[
H_p(\sigma) = -\sum_{i<j} J_{ij} \sigma_i \sigma_j - \sum_i h_i \sigma_i,
\quad \sigma_i = \pm 1
\]

これは「組合せ最適化問題のコスト関数」として解釈できる。

## 2. 量子イジング模型

量子アニーリングでは、同じ構造を持つ「問題ハミルトニアン」を
パウリ行列を用いて

\[
H_p^Q = -\sum_{i<j} J_{ij} \sigma_i^z \sigma_j^z
        -\sum_i h_i \sigma_i^z
\]

として表す。ここで \(\sigma_i^z\) は i 番目の量子ビット上で作用するパウリ Z 行列。

古典イジングのエネルギー関数と、量子イジングの対角成分は 1 対 1 に対応する。

## 3. ドライバーハミルトニアン H_d

量子アニーリングでは、基底状態が簡単なハミルトニアン

\[
H_d = -\sum_i \sigma_i^x
\]

（横磁場項）から始め、時間とともに問題ハミルトニアンへと移行する。

- \(\sigma_i^x\): パウリ X 行列
- 初期状態は \(\sigma^x\) の固有状態（全てのスピンが x 方向に揃った状態）

## 4. 時間依存ハミルトニアン

理想的な量子アニーリングで用いるハミルトニアンは

\[
H(s) = A(s) H_d + B(s) H_p^Q,\quad s \in [0,1]
\]

- 初期: \(s=0\) で \(A(0) \gg B(0)\)（ほぼ \(H_d\)）
- 終了: \(s=1\) で \(B(1) \gg A(1)\)（ほぼ \(H_p^Q\)）

本プロジェクトではまず \(s=1\) のとき、

\[
H(1) \simeq H_p^Q
\]

の **基底エネルギーを数値対角化で求める**。
これは「理想的に量子アニーリングが成功したときの答え」に対応する。

## 5. 古典解法との比較

小さな系（N が小さい）では

- 全探索で求めた最小エネルギー
- 古典 SA（シミュレーテッドアニーリング）が見つけたエネルギー
- 量子イジング \(H_p^Q\) の基底エネルギー

が全て一致するはずであり、
これを確認することで古典 / 量子の定式化が整合していることをチェックできる。
