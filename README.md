# Ising Problem: Classical & Quantum Baseline

本プロジェクトは、任意グラフ上のイジング最適化問題

\[
H_p(\sigma) = -\sum_{i<j} J_{ij}\sigma_i\sigma_j - \sum_i h_i \sigma_i
\]

に対し、

- **古典的手法**  
  - 全探索  
  - シミュレーテッドアニーリング（SA）
- **量子的手法の基礎**  
  - 量子イジング模型 \(H_p^Q\) の行列化と対角化
- **量子アニーリング（QA）の核心となる性質**  
  - 時間依存ハミルトニアン  
    \[
    H(s) = (1-s)H_d + sH_p^Q
    \]
  のスペクトル解析（エネルギーギャップの計算）

を統合的に扱う学習用フレームワークです。

---

## 主要スクリプト

### 1. `main.py`
- 古典手法と量子イジングの **基底エネルギー比較**
- SA の性能チェック

実行:

```sh
python src/main.py
```

---

### 2. `spectrum_scan.py`（H(s) のギャップ計算）
量子アニーリングの難易度を決める  
**最小エネルギーギャップ Δ_min** を計算するスクリプトです。

出力例:
```sh
s E0 E1 gap
0.50 -3.701 -3.403 0.297
...
Minimum gap Δ_min ≈ 0.1409 at s ≈ 0.650
```

ここで

- \(E_0(s)\)：基底状態エネルギー  
- \(E_1(s)\)：第一励起状態エネルギー  
- \(\Delta(s) = E_1(s) - E_0(s)\)

量子アニーリングの成否は **Δ_min がどれだけ小さいか** によって支配されます。

詳細は `docs/spectrum_scan.md` を参照。
